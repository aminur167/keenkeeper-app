# KeenKeeper

KeenKeeper is a responsive friendship reminder app that helps users track friends, remember check-ins, and keep relationships active.

## Tech Stack
React.js, React Router DOM, Vite, Tailwind CSS, DaisyUI, Recharts, LocalStorage

## Features
- Dashboard with banner, summary cards, loading state, and friend cards from `friends.json`.
- Friend details page with profile info, goal cards, and CallTextVideo quick check-ins.
- Timeline with filters and toast notifications for new interactions.
- Friendship analytics page with a Recharts pie chart.
- 404 page and deployment route reload support.

## Run
`npm install` then `npm run dev`


## Build
`npm run build`
